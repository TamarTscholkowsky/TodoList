# TodoList

אפליקציית ניהול משימות (Todo List) מבוססת Web.

## תיאור

TodoList היא מערכת אינטראקטיבית לניהול משימות. ניתן להוסיף, לערוך, למחוק ולסמן משימות כהושלמו, והכל בממשק משתמש נוח ויעיל.

## טכנולוגיות

- **CSS** — עיצוב הממשק
- **C#** — צד שרת (למשל ASP.NET)
- **React** — אינטראקטיביות בצד לקוח
- **Dockerfile** — להרצה בסביבת Docker

## תכונות עיקריות

- הוספת משימות חדשות
- עריכה ועדכון משימות קיימות
- סימון משימות כהושלמו
- מחיקת משימות
- ממשק משתמש רספונסיבי

## התקנה והרצה

1. **שכפול הריפוזיטורי:**
   ```bash
   git clone https://github.com/TamarTscholkowsky/TodoList.git
   cd TodoList
   ```

2. **הרצה עם Docker:**
   ```bash
   docker build -t todolist .
   ```

3. **הרצה מקומית (ללא Docker):**
   - פתח/י את הפרויקט ב־Visual Studio או עורך מתאים ל־C#
   - הרץ/י את הפרויקט (F5 או `dotnet run`)

## שימוש

1. פתח/י את האפליקציה בדפדפן 
2. הוסף/י משימות חדשות דרך שדה ההוספה
3. ערוך/י, מחק/י או סמן/י משימות כהושלמו לפי הצורך
